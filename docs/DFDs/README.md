# DFD Pack

This folder contains the data flow diagrams for UPS PensionsGo.

Files:
- `DFD_Level_0_Context.md` - system context diagram
- `DFD_Level_1_System_Overview.md` - major process decomposition
- `DFD_Level_3_Detailed_Operations.md` - detailed decomposition of the core pension operations and governance backbone

Notes:
- The diagrams are aligned to the implemented application domains: claims and intake, workflow and tasks, registry and documents, payroll and arrears, messaging and content, pensioner self-service, and administration.
- The Level 3 diagram expands the operational core where most pension case movement and governance controls occur.
